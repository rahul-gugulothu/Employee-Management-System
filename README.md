# Employee Management System

## Features

- User Registration
- User Login
- JWT Authentication
- Password Hashing with bcrypt
- Employee CRUD Operations
- MongoDB Atlas Integration
- REST APIs
- Postman Collection for Testing

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- bcryptjs

### Frontend
- React.js (Coming Soon)

## API Endpoints

### Authentication
POST /api/auth/register
POST /api/auth/login

### Employees
POST /api/employees
GET /api/employees
PUT /api/employees/:id
DELETE /api/employees/:id
